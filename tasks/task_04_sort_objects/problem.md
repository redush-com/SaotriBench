# Sort Objects

## Problem
Implement a function that sorts a list of dictionaries by a given key.

## Input
- `items`: a list of dictionaries
- `key`: a string specifying which key to sort by

## Output
- A new sorted list of dictionaries

## Function Signature
```python
def sort_objects(items: list[dict], key: str) -> list[dict]
```

## Notes
- Always return a new list, do not modify the input
- Requirements become stricter in later phases
- The key format and sorting rules may evolve
- Pay attention to feedback about edge cases and key formats
