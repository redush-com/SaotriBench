# Expression Parser

## Problem
Implement a function that evaluates a mathematical expression string.

## Input
- `expression`: a string containing a mathematical expression
- `variables`: optional dict mapping variable names to numeric values

## Output
- The numeric result as a float

## Example
```python
evaluate("2 + 3")     # 5.0
evaluate("2 + 3 * 4") # 14.0
```
